# Triagem-X-Fragil
Sistema web para triagem clínica da Síndrome do X Frágil - Projeto Experiência Criativa
Nome do site = NeuroSaude
  Esse projeto é feito como parte do curso de ciência da computação da Pontifícia Universidade Católica do Paraná, com os alunos em grupo Andrian Carvalho,Matheus Quadros,Luan Orlovski, Bruno Albach,Lucas Prado e Vagner Salviano. O objetivo do projeto é a conscientização e assistência técnica para síndrome do X frágil que afeta milhares de brasileiros. 
  Utilizaremos HTML, CSS e javascript para o front-end do projeto no programa visual studio code, com MYSQL para gerenciar o banco de dados e python para o back-end, utilizando linguagem familiar mais eficiente.
